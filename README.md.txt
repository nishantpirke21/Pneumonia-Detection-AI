# AI X-Ray Pneumonia Detector

A Deep Learning based web application that detects Pneumonia from chest X-ray images using TensorFlow and Streamlit.

## Features
- Upload chest X-ray images
- AI-powered pneumonia detection
- Confidence score visualization
- Real-time prediction

## Tech Stack
- Python
- TensorFlow
- Streamlit
- Deep Learning (CNN)

## Run Project

```bash
pip install -r requirements.txt
streamlit run X-ray.py
